# Product Brief

## Vision

Sirius is a private assistant that combines calendar events, tasks, and health context to help the user plan each day with less friction.

## Primary User

A Mac and iPhone user who wants a calm assistant that can summarize the day, notice constraints, and suggest realistic next actions.

## Core Jobs

- Tell me what is happening today.
- Tell me what I should prepare for.
- Warn me when the day is overloaded.
- Adjust recommendations based on sleep, activity, and recovery signals.

## MVP Features

- Google Calendar daily briefing
- Event preparation checklist
- Manual task capture
- Simple notification summary
- Health integration design, before implementation

## Non-goals For MVP

- Direct medical advice
- Fully autonomous calendar changes
- Reading HealthKit data without explicit iOS permission
