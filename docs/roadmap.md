# Roadmap

## Phase 1: Project Foundation

- Define product scope
- Document integration constraints
- Choose first app surface
- Decide data storage approach

## Phase 2: Calendar MVP

- Set up Google Calendar OAuth
- Fetch today and upcoming events
- Generate daily briefing text
- Add basic reminder output

## Phase 3: Health Context

- Choose HealthKit, export, or Shortcuts integration path
- Read selected health summaries with permission
- Add energy-aware planning suggestions

## Phase 4: Assistant Behavior

- Add task capture
- Add prioritization rules
- Add notification scheduling
- Add feedback loop for better suggestions
